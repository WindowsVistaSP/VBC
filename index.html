在安装Windows Vista 重置后版本之前您可能需要做好以下准备
  1. 请事先准备好驱动、镜像、激活工具，有的版本可能需要提前准备好PE系统（在不同的版本下即使你使用同样的环境，所使用的工具、驱动等都有可能不同）
  2. 一定要在充分了解该版本的详细信息（比如时间炸弹、密钥）后再去安装，建议浏览betawiki或者betaworld
  3. 安装前如果硬盘内有重要文件一定要备份！！！安装系统是需要格式化磁盘的！！！

如果你想在VMware上安装Windows Vista 重置后版本您需要注意
  1. PC端VMware建议使用VMware7-10之间的版本，版本太高或者太低都可能出现不兼容的情况
  2. 50xx-52xx之间的版本建议使用xpdm的显卡驱动，53xx-6000之间的版本建议使用wddm的显卡驱动
  3. 创建虚拟机时建议将兼容性调到VMware 6.5-7.x（除了某些强制要求的版本除外，比如如果您想在Intel CPU上安装5231.x的版本时要将兼容性调到VMware 5.x，否则会蓝屏）
  4. 创建虚拟机时5270以前的版本建议使用Windows xp的默认配置（x64版本要用Windows xp x64Edition），5270以后的版本建议使用Windows Vista的默认配置（x64版本要用Windows Vista x64Edition）
  5. 在进行自己不确定的操作时一定要将虚拟机文件备份
  6. 虚拟机不用时可以选择待机而不是关机，这样下一次使用时可以防止时间炸弹爆炸（如果你使用了激活+拆弹工具并成功的话可以无视这一条）

如何在VMware中开启Windows Vista重置后版本的Aero

5000-5001
  1. 别想啦，这俩版本还没有加入Aero呢

5048
  1. 1.安装xpdm的显卡驱动下载链接
  2. 2.导入以下注册表
  3. Windows Registry Editor Version 5.00
  4. [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\DWM]
  5. "EnableMachineCheck"=dword:00000000
  6. 如果你想要实现其他效果可以点击这里

5098-5112
  1. 1.安装xpdm的显卡驱动下载链接
  2. 2.导入以下注册表
  3. Windows Registry Editor Version 5.00
  4. [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\DWM]
  5. "EnableMachineCheck"=dword:00000000
  6. 3.进PE系统（建议使用Win10PE）将原C:\Windows\System32目下的uxss.exe替换掉（如有保护可以先用管理员权限删除再复制）
  7. PE下载链接
  8. 5098uxss.exe下载链接
  9. 5112uxss.exe下载链接

5212-5219
  1. 1.安装xpdm的显卡驱动下载链接
  2. 2.导入以下注册表
  3. Windows Registry Editor Version 5.00
  4. [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\DWM]

5231
  1. 1.安装xpdm的显卡驱动下载链接
  2. 2.导入以下注册表
  3. Windows Registry Editor Version 5.00
  4. [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\DWM]
  5. "EnableMachineCheck"=dword:00000000
  6. 3.按下Ctrl+Shift+F9关闭Aero，打开设备管理器，将显卡驱动卸载并重启
  7. 4.重启之后再次按下Ctrl+Shift+F9关闭Aero，等到右下角弹出显卡驱动已重新安装的提示时打开调整屏幕分辨率的窗口
  8. 5.将分辨率调的更大一点，然后将鼠标移动到“OK”按钮上（先千万别点那个按钮）
  9. 6.再次按下Ctrl+Shift+F9开启Aero，然后点击“OK”按钮即可

  6. 3.按下Ctrl+Shift+F9关闭Aero，打开设备管理器，将显卡驱动卸载并重启
