4.Aero开启


1.导入以下注册表

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\DWM]

"EnableMachineCheck"=dword:00000000

2.可根据实际情况导入以下键值（Dword）
Dword键值
ForceSoftwareD3D
1 - 强制软件渲染

0 - 硬件渲染
Metal
1 - 不透明Jade样式窗框

0 - build 4050的Glass窗框
UseAlternateButtons
1 - 方形窗口按钮

任何其他值 - build 4050的Glass窗口按钮
ScaleXPercent（警告：使用风险自负！）
0 或 100 （十进制） - 正常水平缩放

1 到 99 范围内的任何十进制值 - 窗口的水平缩小百分比

任何大于 100 的十进制值 - 窗口的水平放大百分比
ScaleYPercent（警告：使用风险自负！）
0 或 100 （十进制） - 正常垂直缩放

1 到 99 范围内的任何十进制值 - 窗口的垂直缩小百分比

任何大于 100 的十进制值 - 窗口的垂直放大百分比

3.在服务（右键计算机——管理）中User Experience Session Management Service的启动类型改为自动（Automatic），注销重启

备注：可以使用Ctrl+Shift+F9开启/关闭Aero
